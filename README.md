# Insurance Analysis Dashboard (Power BI)

## 📊 Project Overview
This project presents an interactive Power BI dashboard built to analyze insurance data. It helps in understanding customer demographics, claim patterns, and policy performance.

## 🔧 Tools Used
- Power BI
- MS Excel (for data preprocessing)

## 📁 Dataset
The dataset includes:
- Customer details
- Policy information
- Claims data

## 📌 Key Features
- Interactive dashboard with filters
- Analysis of claim trends
- Customer segmentation insights
- Policy performance tracking

## 📈 Insights
- Identified high-claim customer segments
- Observed trends in policy renewals
- Highlighted areas of potential risk
